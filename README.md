# 🔗 Blockchain Certificate Authentication

A blockchain-based platform for secure digital certificate authentication and validation using Ethereum.

## 🚀 Features
- Upload PDF certificates and generate SHA-256 hash
- Issue certificates on Ethereum Sepolia blockchain
- Verify certificate authenticity (TRUE/FALSE)
- Tamper-proof and decentralized storage
- MetaMask wallet integration

## 🛠️ Tech Stack
- React.js, HTML, CSS, JavaScript
- Solidity (Smart Contract)
- Ethereum Sepolia Testnet
- Ethers.js
- MetaMask
- Node.js
- Vite

png)

## ⚙️ How to Run
1. Clone the repo
2. cd client
3. npm install
4. npm run dev
5. Open http://localhost:5173
